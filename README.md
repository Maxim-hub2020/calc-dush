# Калькулятор изделий и КП

Адаптивное веб-приложение для расчета душевых и зеркал, сохранения коммерческих предложений и редактирования цен.

Проект работает в браузере и построен на React и Vite.

## Возможности

- Расчет по типу конструкции, размерам, стеклу, фурнитуре, монтажу и доставке.
- Архив КП с поиском, статусами, повторным открытием расчета и удалением.
- Раздел цен для изменения стоимости стекла, фурнитуры, базовых конструкций, услуг, скидки и надбавки за высоту.
- Каталоги цен душевых и зеркал синхронизируются через защищённый API CRM и PostgreSQL.
- На устройстве сохраняется локальная копия цен для расчёта без сети.
- Архив КП текущей версии сохраняется в `localStorage` устройства.

## Локальный запуск

```bash
npm install
npm run dev
```

Локальный адрес по умолчанию: `http://127.0.0.1:5173/`.

## Проверки

```bash
npm run lint
npm run build
```

## Развертывание на Timeweb

Проект подготовлен для размещения рядом с существующей CRM на отдельном поддомене и для автоматической публикации из GitHub. Конфигурация Caddy/Nginx, GitHub Actions, атомарная активация релизов и пошаговая настройка находятся в [`deploy/README.md`](deploy/README.md).

Для общей админки и архива между устройствами следующим шагом нужен backend/API и авторизация.
